# XO Game (C# WinForms)

A Windows Forms implementation of the classic XO (Tic-Tac-Toe) game developed using C# and .NET Framework 4.8, featuring a custom multi-round gameplay mechanic.

### Features

- Two-player gameplay
- Interactive graphical interface
- Turn-based player switching
- Winner detection system
- Multi-round gameplay
- Automatic board continuation when no winner is found
- Random row, column, or diagonal reset between rounds
- Round counter tracking
- Game restart functionality

### Concepts Practiced

- Windows Forms Development
- Event-Driven Programming
- Button / PictureBox Controls
- Labels and Dynamic UI Updates
- Game State Management
- Conditional Logic
- Randomization
- User Interface Design
- Resource Management

### Technologies

- C#
- .NET Framework 4.8
- Windows Forms (WinForms)

## Purpose

This project was developed to practice building interactive desktop applications using WinForms while implementing game logic, event handling, state management, and custom gameplay mechanics beyond the traditional Tic-Tac-Toe rules.
